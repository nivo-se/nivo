# ANALYST_WORKBENCH_UX.md

## Analyst Workflow

1. Select company
2. Run analysis
3. Monitor pipeline progress
4. Inspect generated report
5. Review verification panel
6. Edit competitors or assumptions
7. Trigger recompute
8. Compare report versions

## Key UX Requirements

- Transparent pipeline progress
- Easy navigation between report sections
- Visible verification status
- Quick competitor editing
- Clear report version history
